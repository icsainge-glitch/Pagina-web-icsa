# **App Name**: ICSA Solutions

## Core Features:

- Services Showcase: Clearly display all communication engineering, security system, electrical study, and network services offered by ICSA Ingeniería S.A. with detailed descriptions.
- About Us & Contact: Provide company information, mission, vision, and comprehensive contact details including location, phone, and a direct inquiry form.
- Project Portfolio: Showcase a curated selection of past projects, highlighting ICSA's expertise in infrastructure, cabling, and systems installations.
- AI-Assisted Inquiry Tool: An interactive form where users describe their project needs, and an AI tool suggests relevant ICSA services or asks for more information before generating a preliminary inquiry.
- Supplier Information: A section detailing the types of equipment and communication data accessories supplied by ICSA.

## Style Guidelines:

- Primary color: A professional and trustworthy medium-dark blue (#226EE6), signifying expertise and reliability in technology and engineering. (HSL: 220, 70%, 45%)
- Background color: A very light, subtle bluish-gray (#D9E2EF), providing a clean and understated canvas. (HSL: 220, 15%, 93%)
- Accent color: A vibrant, clear cyan (#14C4E0), used sparingly for call-to-action buttons, key highlights, and interactive elements to draw attention. (HSL: 190, 80%, 55%)
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, and highly legible experience suitable for technical and professional content.
- Use clean, geometric, and modern line icons to visually represent services and enhance navigation, consistent with a tech and engineering aesthetic.
- Implement a clean, grid-based, responsive layout with clear sections for services, projects, and contact information, ensuring accessibility across devices.
- Subtle and professional animations for navigation transitions, button hovers, and element loading to create a polished user experience without distractions.
